# Riding Cities – Mise à jour HTML

Projet réalisé dans le cadre de ma formation de développeur web.  
Il s'agit d'un exercice d’intégration HTML basé sur une maquette fournie.

## Contexte

Dans un scénario de stage fictif chez l’agence Webonzaï, j’ai reçu pour mission de mettre à jour le site de Riding Cities, une association de skate.

## Objectifs

- Ajouter une section "Les membres fondateurs" avec leurs noms et photos.  
- Modifier la section de téléchargement en ajoutant un titre et deux boutons : un pour le planning enfants et un pour le planning adultes.

## Maquette

[Voir la maquette Figma](https://www.figma.com/design/EGLtVGfH0H7S4068Fl5EOO/Maquette-desktop-Riding-Cities?node-id=0-1&p=f&t=kxbFXSIUbOGGFHiG-0)

## Technologies utilisées

- HTML5  
- CSS3 (déjà présent, utilisé sans modification)  
- Figma (consultation de la maquette)  
- Visual Studio Code
